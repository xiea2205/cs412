# Database Deployment Solution

## Option 1: Copy Local Database to Server

# 1. On your LOCAL machine, ensure database is ready:
pipenv run python manage.py migrate
pipenv run python manage.py create_sample_profiles

# 2. Copy db.sqlite3 to server (run this from your local machine):
scp db.sqlite3 xiea@csa2.bu.edu:webapps/django/

# Alternative: Use git to include the database
git add db.sqlite3
git commit -m "Add database with mini_insta tables and sample data"
git push origin main

# Then on server:
git pull
